E-shop

Description du projet

E-shop est une application de commerce électronique développée en utilisant le framework Spring Boot. Elle permet aux utilisateurs de parcourir une large gamme de produits, d'ajouter des articles à leur panier et de passer des commandes en ligne. L'application offre une interface conviviale et sécurisée pour les transactions, ainsi que des fonctionnalités de gestion des stocks et de suivi des commandes pour les administrateurs.

Technologies utilisées

  • Spring Boot
  • Java
  • Thymeleaf
  • MySQL
  • Bootstrap
