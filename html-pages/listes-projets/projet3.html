Dashboard Analytics

Description du projet

Dashboard Analytics est une application de visualisation de donnees developpee en utilisant le framework React. Elle permet aux utilisateurs de visualiser et d'analyser des donnees complexes a travers des graphiques et des tableaux de bord interactifs. L'application offre une interface conviviale et personnalisable pour les transactions, ainsi que des fonctionnalites de gestion des stocks et de suivi des commandes pour les administrateurs.

Technologies utilisees

  • React
  • JavaScript
  • Redux
  • Node.js
  • Express
